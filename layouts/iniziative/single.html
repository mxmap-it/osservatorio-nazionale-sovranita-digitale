{{ if $isEN }}The MxMap family in Europe{{ else }}La famiglia MxMap in Europa{{ end }}

{{ if $isEN }}Projects that use the same methodology as the Observatory — the analysis of the MX records of public administrations' domains — to map email providers and their jurisdiction. The original project is MxMap.ch, created by David Huser for Swiss municipalities; the up-to-date list of instances is maintained on the project's GitHub repository.{{ else }}Progetti che usano la stessa metodologia dell'Osservatorio — l'analisi dei record MX dei domini delle amministrazioni pubbliche — per mappare i provider email e la loro giurisdizione. Il progetto originale è MxMap.ch, creato da David Huser per i comuni svizzeri; l'elenco aggiornato delle istanze è mantenuto sul repository GitHub del progetto.{{ end }}

{{ if $isEN }}Other digital sovereignty measurement initiatives{{ else }}Altre iniziative di misurazione della sovranità digitale{{ end }}

{{ if $isEN }}Initiatives that measure digital sovereignty from angles that are different from, and complementary to, the Observatory.{{ else }}Iniziative che misurano la sovranità digitale da angolazioni diverse e complementari rispetto all'Osservatorio.{{ end }}

sovranita-digitale.it {{ if $isEN }}Italy{{ else }}Italia{{ end }}

{{ if $isEN }}An independent Italian civic initiative that collects and analyses multiple databases and public sources (cloud, DNS, ASN, TLS, lock-in, interoperability) to produce a multidimensional sovereignty analysis of the PA — with maps and indices per body. A broader approach than the Observatory's focus on email, and complementary.{{ else }}Iniziativa civica indipendente italiana che raccoglie e analizza plurime basi dati e fonti pubbliche (cloud, DNS, ASN, TLS, lock-in, interoperabilità) per produrre un'analitica di sovranità multidimensionale della PA — con mappe e indici per ente. Approccio più ampio rispetto al focus dell'Osservatorio sulla posta elettronica, e complementare.{{ end }}

sovranita-digitale.it

pa.zerozone.it {{ if $isEN }}Italy{{ else }}Italia{{ end }}

{{ if $isEN }}An Italian initiative that catalogues and monitors the websites of the PA bodies registered in IndicePA, with screenshots, domain analysis and MX record analysis (including the use of providers such as Google). A complementary snapshot of public digital infrastructure, on the web side as well as email.{{ else }}Iniziativa italiana che cataloga e monitora i siti web delle PA registrate nell'IndicePA, con screenshot, analisi dei domini e dei record MX (incluso l'uso di provider come Google). Una fotografia complementare dell'infrastruttura digitale pubblica, sul versante web oltre che email.{{ end }}

pa.zerozone.it

EDRIX — European Digital Resilience Index

{{ if $isEN }}A 0–10 index of the digital dependence of the 27 EU member states, broken down into 4 pillars — including public-sector hosting sovereignty, the area closest to ours. Data from public and scriptable sources.{{ else }}Indice 0–10 sulla dipendenza digitale dei 27 Stati membri UE, articolato in 4 pilastri — tra cui la sovranità dell'hosting del settore pubblico, l'ambito più vicino al nostro. Dati da fonti pubbliche e scriptabili.{{ end }}

edrix.eu

Nextcloud — Digital Sovereignty Index (DSI)

{{ if $isEN }}Measures the adoption of self-hosted collaboration tools (files, video conferencing, groupware) in around 60 countries, as an indicator of digital independence.{{ else }}Misura l'adozione di strumenti collaborativi self-hosted (file, videoconferenza, groupware) in circa 60 paesi, come indicatore di indipendenza digitale.{{ end }}

nextcloud.com

EU Cloud Sovereignty Framework (SEAL)

{{ if $isEN }}The European Commission's official framework for assessing the sovereignty of cloud services in public procurement, with SEAL 1–4 assurance levels (from data sovereignty to full technological autonomy).{{ else }}Quadro ufficiale della Commissione Europea per valutare la sovranità dei servizi cloud nel procurement pubblico, con livelli di assurance SEAL 1–4 (dalla sovranità dei dati all'autonomia tecnologica piena).{{ end }}

commission.europa.eu
