{{ if $isEN }}You don't need to be a technology expert to act. This kit gathers practical tools to hold your administration to account and spread awareness of digital sovereignty.{{ else }}Non serve essere esperti di tecnologia per agire. Questo kit raccoglie strumenti pratici per chiedere conto alla tua amministrazione e diffondere consapevolezza sulla sovranità digitale.{{ end }}

{{ if $isEN }}Change starts from the ground up{{ else }}Il cambiamento parte dal basso{{ end }}

{{ if $isEN }}When a citizen asks their municipality where their emails end up, the administration is forced to ask itself the question. Civic pressure, multiplied, becomes institutional awareness and then action.{{ else }}Quando un cittadino chiede al proprio Comune dove finiscono le sue email, l'amministrazione è costretta a porsi la domanda. La pressione civica, moltiplicata, diventa consapevolezza istituzionale e poi azione.{{ end }}

{{ if $isEN }}What you can do, concretely{{ else }}Cosa puoi fare, concretamente{{ end }}

{{ if $isEN }}1. Check your public body{{ else }}1. Verifica il tuo ente{{ end }}

{{ if $isEN }}Search for your municipality, your local health authority or your school in the Observatory's data and find out which provider handles their email and under which jurisdiction.{{ else }}Cerca il tuo Comune, la tua ASL o la tua scuola nei dati dell'Osservatorio e scopri quale provider gestisce le loro email e sotto quale giurisdizione.{{ end }}

{{ if $isEN }}2. Ask the right question{{ else }}2. Fai la domanda giusta{{ end }}

{{ if $isEN }}Write to your municipality (certified email, town council, civic access request) and ask which email provider it uses, where the data is stored and whether a jurisdictional risk assessment has been carried out.{{ else }}Scrivi al tuo Comune (PEC, consiglio comunale, accesso civico) e chiedi quale provider email usa, dove sono archiviati i dati e se è stata fatta una valutazione del rischio giurisdizionale.{{ end }}

{{ if $isEN }}3. Spread the data{{ else }}3. Diffondi i dati{{ end }}

{{ if $isEN }}Share the results on social media and with the local press. A story about "where our municipality's emails end up" has a strong local impact.{{ else }}Condividi i risultati sui social e con la stampa locale. Una notizia su "dove finiscono le email del nostro Comune" ha forte impatto sul territorio.{{ end }}

{{ if $isEN }}4. Bring the issue to the council{{ else }}4. Porta il tema in consiglio{{ end }}

{{ if $isEN }}Propose to a town councillor you trust a motion or a formal question on the public body's digital sovereignty, using the data as a basis.{{ else }}Proponi a un consigliere comunale di tua fiducia una mozione o un'interrogazione sulla sovranità digitale dell'ente, usando i dati come base.{{ end }}

{{ if $isEN }}Civic access request template{{ else }}Modello di richiesta di accesso civico{{ end }}

{{ if $isEN }}You can use civic access (FOIA, Legislative Decree 33/2013) to obtain information from the public body. Here is a sample text to adapt:{{ else }}Puoi usare l'accesso civico (FOIA, D.Lgs. 33/2013) per ottenere informazioni dall'ente. Ecco un testo di esempio da adattare:{{ end }}

{{ if $isEN }}Subject: Generalised civic access request (art. 5, para. 2, Legislative Decree 33/2013) The undersigned [name] requests to know: 1. which provider handles the public body's email service; 2. in which country the email data is stored; 3. whether the provider is subject to non-EU legislation that would allow access by foreign authorities (e.g. the CLOUD Act); 4. whether a data protection impact assessment (DPIA) has been carried out in relation to that service. Yours faithfully, [name, date]{{ else }}Oggetto: Richiesta di accesso civico generalizzato (art. 5, c. 2, D.Lgs. 33/2013) Il/la sottoscritto/a [nome], chiede di conoscere: 1. quale fornitore gestisce il servizio di posta elettronica dell'ente; 2. in quale Paese sono archiviati i dati di posta; 3. se il fornitore è soggetto a normative extra-UE che ne consentano l'accesso da parte di autorità straniere (es. CLOUD Act); 4. se è stata effettuata una valutazione d'impatto sulla protezione dei dati (DPIA) relativa a tale servizio. Distinti saluti, [nome, data]{{ end }}

{{ if $isEN }}This template is provided for informational purposes and must be adapted to the specific case. It does not constitute legal advice.{{ else }}Questo modello è fornito a titolo informativo e va adattato al caso concreto. Non costituisce consulenza legale.{{ end }}
