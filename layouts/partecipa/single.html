Come iniziare
  1. Fai un fork del repository GitHub
  2. Crea un branch per le tue modifiche
  3. Apri una Pull Request descrivendo il tuo contributo

Non sai usare GitHub? Nessun problema — puoi anche aprire una issue per suggerimenti e segnalazioni.
